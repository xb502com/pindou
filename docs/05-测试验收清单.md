## 2. 围圈 mask

## 6. 回归禁区

以下情况一旦出现，必须阻断发布：

1. 8 字圈被当成错误拦截。
2. 相切/共享边界被要求拆开。
3. 圈套圈没有提示，导致 mask 语义不清。
4. 深色豆仍使用深灰序号导致看不清。
5. SVG/PDF 只是嵌入一整张位图。
6. 生产环境从外部 CDN 拉取关键 WASM/wheel。
7. 用户图片发生上传请求。
