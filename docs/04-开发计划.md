# 拼豆图生成器（Pingdou）— 开发计划 v0.7

> 目标：按可验证里程碑开发，优先满足拼豆用户真实需求：本地隐私、稳定打开、好圈范围、按颜色拼装、矢量清晰打印。

---

## 1. 里程碑总览

| 阶段 | 名称 | 目标 | 产出 |
|------|------|------|------|
| M0 | 工程骨架 | 静态目录与 stlite hello world | `index.html`、自托管资源目录、基础 app |
| M1 | 图片与网格 | 上传、裁剪、网格预览 | 可本地预览 W×H 方格 |
| M2 | 容错围圈 | 8 字/叠边合法，圈套圈拦截 | `mask_cv.py` + 交互提示 |
| M3 | 颜色与标序 | 色板量化、分色局部编号 | `quantize.py`、`components.py`、图例 |
| M4 | 图纸预览 | 总览、单色高亮、自适应文字色 | 可用于拼装的屏幕图纸 |
| M5 | 导出 | SVG/PDF/PNG/CSV | 用户可打印/分享/采购 |
| M6 | 部署与广告 | Cloudflare Pages、自托管验收、广告位 | 可公开上线 |

---

## 2. 任务拆解

### M0：工程骨架

- [ ] 创建 `public/index.html`，实现 CSS loading。
- [ ] 创建 `public/assets/stlite/`、`pyodide/`、`wheels/`、`app/`。
- [ ] 实现 stlite 加载 `assets/app/main.py`。
- [ ] 加入资源完整性检查脚本。
- [ ] 本地静态服务器打开并显示 hello world。

验收：断开外部 CDN，仅访问本站静态文件仍能打开 hello world。

### M1：图片、裁剪、网格

- [ ] 上传 PNG/JPG/WebP，读取为 PIL/NumPy。
- [ ] 裁剪参数 UI。
- [ ] 网格尺寸预设与自定义。
- [ ] 网格叠加预览。
- [ ] `grid.py` 完成图像格子坐标映射。

验收：用户能看到裁剪图上的方格，不做颜色量化。

### M2：容错围圈

- [ ] 实现边界格绘制、擦除、清空。
- [ ] 拖拽中淡蓝预览，不弹错。
- [ ] Mouse Up / 500ms debounce 静默检测。
- [ ] `mask_cv.py` 实现：空边界、单圈、多圈、8 字、共享边界、外部毛刺。
- [ ] 实现圈套圈检测与高亮提示。
- [ ] 增加单元测试用例。

验收：8 字圈不报错；圈套圈报错；外部毛刺不误报未闭合。

### M3：颜色转换与标序

- [ ] 色板 JSON 加载与 Lab 预计算。
- [ ] 点位 RGB 均值。
- [ ] ΔE 最近色。
- [ ] 每色 `connectedComponents(8)`。
- [ ] TB-LR 重映射。
- [ ] 短字母前缀与色板 code 前缀。
- [ ] 颜色用量和块清单。

验收：同色相连区域同号，不同颜色分别从 1 开始。

### M4：拼装图纸 UX

- [ ] 彩色总览图。
- [ ] 自适应序号色：深色豆浅字，浅色豆深字。
- [ ] 颜色勾选列表：色块、名称、色号、颗数、块数。
- [ ] 单色专注 / 全部显示。
- [ ] 图例固定展示。

验收：用户能按颜色拿豆并快速找到该颜色所有区域。

### M5：导出

- [ ] SVG：`rect + text + legend`。
- [ ] CSV：颜色汇总 + 块明细。
- [ ] PNG：当前视图高清导出。
- [ ] PDF：至少支持浏览器打印 SVG；后续升级矢量 PDF 直出。
- [ ] 当前高亮状态导出。

验收：SVG/PDF 放大不糊；CSV 能指导采购和拼装。

### M6：部署与广告

- [ ] Cloudflare Pages 发布。
- [ ] `_headers` 缓存策略。
- [ ] 生产环境 Network 检查无外部关键依赖。
- [ ] 广告 loader 异步挂载。
- [ ] 广告失败降级。

验收：Cloudflare Pages 上完整流程可用，广告不影响主功能。

## 3. 风险与缓解

| 风险 | 影响 | 缓解 |
|------|------|------|
| OpenCV wheel 体积大/兼容性 | 首屏慢或加载失败 | 锁定可用版本；必要时以 JS OpenCV 或轻量 cv2 wasm 替代 |
| stlite 初始化慢 | 首次体验差 | loading 动画、文案、缓存、资源压缩 |
| 围圈边缘 case 多 | 用户范围不准 | 测试矩阵覆盖 8 字、叠边、毛刺、圈套圈 |
| PDF 直出复杂 | 开发延期 | MVP 用 SVG + 浏览器打印，后续矢量 PDF 库 |
| 国内网络波动 | 白屏 | 全量自托管，不使用运行时外部 CDN |
