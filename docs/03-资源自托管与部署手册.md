# 拼豆图生成器（Pingdou）— 资源自托管与部署手册 v0.7

> 目标：确保国内用户稳定访问，生产运行时不依赖 GitHub/jsDelivr/unpkg/PyPI 等外部动态资源。

---

## 1. 自托管清单

| 类别 | 文件 | 路径 | 是否 P0 |
|------|------|------|---------|
| stlite | runtime JS/CSS | `public/assets/stlite/` | 是 |
| Pyodide | `pyodide.js` | `public/assets/pyodide/pyodide.js` | 是 |
| Pyodide | `pyodide.asm.wasm` | `public/assets/pyodide/pyodide.asm.wasm` | 是 |
| Pyodide | `python_stdlib.zip` | `public/assets/pyodide/python_stdlib.zip` | 是 |
| Pyodide | lock/metadata | `public/assets/pyodide/` | 是 |
| wheels | NumPy | `public/assets/wheels/` | 是 |
| wheels | Pillow | `public/assets/wheels/` | 是 |
| wheels/vendor | OpenCV WASM/cv2 | `public/assets/wheels/` 或 `public/assets/vendor/opencv/` | 是 |
| palettes | 拼豆色板 JSON | `public/assets/palettes/` | 是 |
| fonts | 导出字体 | `public/assets/fonts/` | 建议 |
| ads | 广告 loader | `public/assets/vendor/ads/` | 可选 |

---

## 2. 生产禁止项

- 禁止页面运行时从以下域名拉取关键资源：
  - `cdn.jsdelivr.net`
  - `unpkg.com`
  - `github.com`
  - `raw.githubusercontent.com`
  - 远程 PyPI / wheels 源
- 禁止依赖外部 CDN 成功后应用才能启动。
- 禁止用户图片上传到任何业务服务器、广告服务或统计服务。

---

## 3. 建议构建脚本

```text
scripts/
├── download_pyodide_assets.py   # 固定版本下载 pyodide 核心资源
├── vendor_wheels.py             # 固定版本下载 wheels 并校验 hash
├── verify_assets.py             # 检查 public/assets 是否完整
├── build_static.py              # 生成 dist/public
└── smoke_check_network.py        # 扫描 index.html/app 是否引用外部 CDN
```

### 3.1 资源 manifest

```json
{
  "pyodide_version": "固定版本号",
  "stlite_version": "固定版本号",
  "wheels": [
    {"name": "numpy", "version": "固定版本号", "sha256": "..."},
    {"name": "pillow", "version": "固定版本号", "sha256": "..."},
    {"name": "opencv", "version": "固定版本号", "sha256": "..."}
  ]
}
```

所有资源必须锁版本和 hash，避免“今天能跑，明天 CDN 包变了”。

## 4. Cloudflare Pages 设置

### 4.1 构建产物

- 发布目录：`public/` 或 `dist/`。
- 不需要后端函数。
- 如后续使用 Pages Functions，仅限轻量统计/配置，不得处理图片。

### 4.2 `_headers`

```text
/index.html
  Cache-Control: no-cache

/assets/pyodide/*
  Cache-Control: public, max-age=31536000, immutable

/assets/wheels/*
  Cache-Control: public, max-age=31536000, immutable

/assets/stlite/*
  Cache-Control: public, max-age=31536000, immutable

/assets/app/*
  Cache-Control: public, max-age=3600

/assets/palettes/*
  Cache-Control: public, max-age=86400
```

### 4.3 DDoS 与缓存

- 主站全部静态化，Cloudflare 边缘节点可直接缓存。
- 没有源站算法服务，攻击流量不会放大服务器 CPU 成本。
- 可开启 Cloudflare Bot Fight / WAF / Rate Limit，但不要影响静态资源正常缓存。

---

## 5. 广告挂载规范

- 广告脚本异步加载，不阻塞 `stlite`。
- 广告容器预留尺寸，避免布局跳动。
- 广告脚本失败不影响主应用。
- 广告脚本不得读取用户图片、canvas、工程 JSON。

```html
<aside class="ad-slot ad-top"></aside>
<script async src="/assets/vendor/ads/ads-loader.js"></script>
```

---

## 6. 上线前网络验收

1. 清空缓存首次访问，页面显示 loading。
2. DevTools Network 过滤 `pyodide|wasm|whl|stlite`，所有 URL 均为本站域名。
3. 屏蔽 GitHub/jsDelivr/unpkg 后，应用仍可完整启动。
4. 上传图片、生成图纸、导出文件过程中无图片上传请求。
5. 广告域名失败时，主流程仍可使用。
