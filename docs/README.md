## v0.7 关键业务原则

1. 纯前端静态网页：用户图片不上传，算法全部本地 WASM 执行。
2. 围圈是为了圈定参与染色/标序/导出的拼豆范围，不是 CAD 拓扑编辑器。
3. 8 字圈、相切、相交、共享边界、重复涂边都应容错接受。
4. 唯一需要强拦截的是圈套圈/完全嵌套，因为它会引入挖洞语义。
5. 默认按颜色前缀 + 局部连通块编号，服务用户按颜色集中拼装。
6. SVG/PDF 是主导出格式，保证手机放大和物理打印清晰。
